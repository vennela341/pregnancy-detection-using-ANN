# Pregnancy Prediction using ANN

## Project Description
This project predicts the possibility of pregnancy-related outcomes using an Artificial Neural Network (ANN) model.

## Dataset
The model is trained using a medical dataset containing features such as glucose level, blood pressure, BMI, age, and insulin level.

## Technologies Used
Python
NumPy
Pandas
Scikit-learn
TensorFlow
Matplotlib

## Steps in the Project
1. Data Collection
2. Data Preprocessing
3. Feature Scaling
4. ANN Model Training
5. Prediction

## How to Run the Project
1. Install required libraries
2. Run the Jupyter Notebook
3. Input values to get prediction
