# Pregnancy Data Analysis Project

## Project Description
This project analyzes pregnancy-related data to understand the factors that affect baby birth weight. The dataset contains information about mothers such as age, height, weight, gestation period, and smoking habits. By analyzing these features, the project studies their influence on the baby's birth weight.

## Dataset
The dataset used in this project is **babies.csv**.

### Dataset Features
- case – Case identification number
- bwt – Baby birth weight
- gestation – Duration of pregnancy in days
- parity – Number of previous pregnancies
- age – Age of the mother
- height – Height of the mother
- weight – Weight of the mother
- smoke – Smoking habit of the mother (0 = No, 1 = Yes)
